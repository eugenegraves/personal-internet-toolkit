The Architectural Thesis

Not every UI surface deserves the same framework. A keystroke-heavy editor benefits from compile-time reactivity (Svelte). A virtualized list needs controlled rendering (React). A form submission is perfectly served by hypermedia (HTMX). A tree explorer needs no framework at all.

AbsoluteJS acts as the orchestration layer — routing, SSR, and build tooling — while each tool loads only the framework it needs. No monolithic bundle. No wasted runtime.
